

| Sheet Name        | Description                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------ |
| `README`          | Overview of file contents, purpose, and instructions                                       |
| `Data_Raw`        | Unprocessed macroeconomic data collected from official sources                             |
| `QuarterlyRealGDP`| Real GDP calculated quarterly                                                              |
| `HP Filter`       | Applies the HP Filter to Japan’s real GDP data to extract its trend and cyclical components|
| `Okun's ANOVA`    | Regression analysis between unemplyment rate and GDP growth rate                           |
| `PC ANOVA`        | Regression anlysis between inflation rate and change in unemployment rate                  |
| `IS ANOVA`        | Analysis between the HP filtered log GDP and HP filtered real interest rate                |
| `Labor Force`     | Analysis of the employment rate and the share of older workers (55-64)                     |
| `Productivity ULC`| Analysis of Unit Labor Cost and the share of older workers (55-64)                         |
| `Abenomics`       | Visualization of public investments/ policy rate and real GDP, pre and post Abenomics      |


📌 Notes & Assumptions
Currency values are in Japanese Yen (JPY).
GDP data is seasonally adjusted for the stated quarters.
